5.创建一个目录用于存放这个解压好的jdk

6.将解压好的jdk目录移动到创建好的目录中去

7.配置环境变量

先执行文件编辑命令vim /etc/profile
然后将以下2行命令复制插入到编辑文件的末尾
export JAVA_HOME=/usr/local/src/java/jdk1.7.0_71
export PATH=$JAVA_HOME/bin:$PATH
保存退出即可

8.让修改的配置生效
进入安装jdk的bin目录，执行以下命令：
source /etc/profile

安装tomcat

1.将我们需要安装tomcat上传到Linux系统中去

  • alt+p打开文件上传的窗口 将apache-tomcat-7.0.57.tar.gz拖拽进去

2.解压这个tomcat

3.创建一个目录用于存放这个解压好的tomcat(之前已经创建好了)

4.将解压好的tomcat目录移动到创建好的目录中去

5.配置tomcat端口对外公开【能够在windows系统中访问Linux系统里面的tomcat软件】
执行以下指令：

在-A INPUT -m state --state NEW -m tcp -p tcp --dport 22 -j ACCEPT后面按下回车键
将上面的这一行复制粘贴，修改22为8080
保存退出

6.重启防火墙

7.进入安装好的tomcat的bin目录

8.启动tomcat

在bin目录中执行 ./startup.sh
如果看到：Tomcat started. 说明tomcat启动成功！

9.测试：
在windows系统中，打开浏览器，输入：http://192.168.59.132:8080
回车后如果看到了tomcat猫，说明tomcat安装并且配置OK

10.关闭tomcat
在bin目录中执行./shutdown.sh

Redis安装

1.安装Redis的依赖
执行指令：

2.将我们需要安装redis上传到Linux系统中去
alt+p打开文件上传的窗口 将redis-3.0.0.tar.gz拖拽进去

3.解压这个redis

4.创建一个目录用于存放这个解压好的redis

5.将解压好的redis目录移动到创建好的目录中去

6.编译redis
注意：一定要确保依赖已经安装OK
先得进入目录：cd /usr/local/src/java/redis/redis-3.0.0
然后执行指令：make

7.进行安装
执行指令；make PREFIX=/usr/local/src/java/redis install
注意：PREFIX必须是大写！其值为redis安装的目录

8.移动文件【方便后面启动redis】
首先的进入目录：cd /usr/local/src/java/redis/redis-3.0.0
执行以下指令(移动到redis目录中去)：
cp redis.conf /usr/local/src/java/redis

9.启动redis服务
先得修改配置文件
执行指令：vim /usr/local/src/java/redis/redis.conf
在文件的末尾添加： # daemonize yes
保存退出
启动：
进入redis即可
cd /usr/local/src/java/redis
执行指令：
./bin/redis-server ./redis.conf

如果看到了：The server is now ready to accept connections on port 6379 说明启动成功

10.停止redis服务
进入redis目录，执行指令：./bin/redis-cli shutdown
如果看到了：Redis is now ready to exit, bye bye...说明redis停止OK

11.连接redis
得先启动redis服务！
克隆一个窗口
进入目录cd /usr/local/src/java/redis
执行指令：./bin/redis-cli -h 192.168.59.132 -p 6379 【手敲，复制无用】
如果看到了：192.168.59.132:6379> 说明连接OK
