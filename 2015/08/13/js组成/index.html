1.ECMAScript：js核心语法

一、数据类型 ：

  1. 原始类型： String Number Boolean Null Undefined
  1. 引用类型 new Date() Math Array

二、函数：

  1. 普通函数

function 函数名() {}

  1. 匿名函数

var 函数名 = function (){}

三、事件

onclick（点击事件）

onload（加载完成事件）

onsubmit（表单提交事件）（form）

onfocus（获取焦点）

onblur（失去焦点）

onchange（改变事件）

var inpObj = document.getelementById(“inp”);

inpObj.onclick = function（）{}

2.BOM（浏览器对象模型）

  • window

  • Window 对象表示浏览器中打开的窗口,如果文档包含框架（frame 或 iframe 标签），浏览器会为 HTML 文档创建一个 window 对象，并为每个框架创建一个额外的 window 对象。

    ​ 常用属性:

    ​ 通过它获取其他的四个对象

    ​ eg:

    ​ window.history == history

    ​ 注意:

    ​ 使用window的对象或属性时,window可以省略.

    ​ 定时器:

    ​ 弹出框:

    ​ 警告框

    ​ alert();

    ​ 对话框:

    ​ prompt();

    ​ 确认框:

    ​ confirm();

    ​ 扩展:打开和关闭

    ​ open(url);

    ​ close();

    ​ history

    ​ 常用方法:

    ​ forward(); 前进到指定页面

    ​ back(); 后退

    ​

    ​ ★go(num); num前进或后退几个页面

    ​ ★location

    ​ 属性:

    ​ href

    ​ location.href 获取当前页面的url

    ​ location.href=”url”; 调向指定页面

    ​

    ​ navigator

    ​ screen

  • 定时器

  • 定时器:

    ​ 周期:

    ​ var interId = setInterval(函数名称,毫秒值);

    ​ var interId = setInterval(“函数名称(参数列表)”,毫秒值);

    ​

    ​ clearInterval(interId);

    ​ 单次:

    ​ var timeId = setTimeout(函数名称,毫秒值);

    ​ var timeId = setTimeout(“函数名称(参数列表)”,毫秒值);

    ​

    ​ clearTimeout(timeId);

3.DOM（文档对象模型）

  • DOM:

    ​ Element:元素(标签)

    ​ Attribute:属性

    ​ Event:事件

  • document:

    ​ 元素:(element)

  • 查:(获取)

document.getElementById(“id属性的值”); 通过id获取一个元素对象

document.getElementsByName(“name属性的值”); 通过name属性获取多个元素对象 元素对

document.getElementsByTagName(“标签名称”); 通过标签名称获取一种元素对象 元素对象数组

​ document.getElementsByClassName(“class属性的值”); 通过class获取一类元素对象 元素对象数组

  • 增:

    document.createElement(“标签名”);

创建元素节点(创建一个标签)

document.createTextNode(“值”);

创建文本节点(创建一个值)

​ 元素节点.appendChild(文本节点); ‘

给元素节点添加文本信息

​ 元素.appendChild(子元素对象);

将元素节点追加到指定标签中

​ 元素.innerHTML = “

文本内容

“;

删:

元素对象.remove(); 删除自己(自杀)

属性:(attribute)

​ ★简单操作方法:

​ 元素.属性名称; 获取指定属性的值

​ 元素.属性名称 = “属性值”; 设置指定属性的值

​ 标准操作:

​ 查(获取):

​ 元素.getAttribute(属性名称);

​ 增:

​ 元素.setAttribute(“属性名称”,”属性值”);

​ 删:

​ 元素.removeAttribute(“属性名称”);

​ 标签体:

​ 元素.innerHTML;

​ 元素.innerHTML = “标签体”;
