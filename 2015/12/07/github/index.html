三、github的介绍

1、什么是github:管理开源项目的网站

2、 github 地址：https://github.com/

3、Github教程 - 廖雪峰的官方网站

4、Git与Github 的关系

  • git 是一个分布式管理工具
  • github 是为开发者提供git 仓库托管服务

四、github 安装 和 git-bash 的基本使用

1、关于git-bash 的下载：注意64位电脑下载64位的，如果32位的要使用本电脑下载

  • git-bash 就是一个命令行工具
  • 地址是：https://git-for-windows.github.io/

2、git-bash 安装注意事项

  • 最好不要更改安装地址
  • 注意自己的电脑位数(32位或者64位)
  • 安装完成后进行测试 git --version,显示版本号则表示安装成功

五、部署静态Blog

1、 通过hexo 来部署 网站

2、 所需要的环境

  • Node 环境、 npm 全局安装 、github 账号

    3、安装hexo 环境 npm install -g hexo-cli

  • 检查是否安装成功：hexo --version,出现版本号则表示安装成功

4、博客初始化 hexo init blog

六、博客的发布

1、修改配置文件

  • 找到blog 文件夹，找到config 文件，找到 deploy 、 type: 设置成键值对的形式：;
  • type: git / repo: 生成的io 文件，在文件的前面加上用户名：密码@加github地址
  • 在项目的跟目录下（也就是说blog 下执行gitbash 命令 npm install hexo-deployer-git --save）
  • 在上面的基础上执行 hexo deploy --generate 就可以提交成功了
  • 通过用户名加github.io 就可以访问了https://hostpink.github.io/
