单人开发

  • 在计算机的任意盘符创建一个文件夹：d:/gitdemo/tangseng
  • 进入tangseng目录下进行初始化：git init
  • 在tangseng目录下创建一个test.java文件：touch test.java
  • 查看文件的状态：git status(查看所有的文件状态） git status filename(查看指定的文件状态)
    • 文件为红色：未被git管理
      • 文件为绿色：已被git管理
  • 将文件提交到暂存态：git add test.java(指定的文件) git add .(提交所有的文件)
  • 将文件提交到本地班级库：git commit -m “注释内容”
  • 查看操作记录：git log
